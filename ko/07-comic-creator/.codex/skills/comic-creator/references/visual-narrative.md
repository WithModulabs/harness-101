# Visual Narrative

Use for dialogue, sound effects, speech bubbles, and editing directions.

## Principle

In comics, much of the story is carried by drawing, composition, expression, sound effects, and silence. Remove dialogue that repeats what the panel already shows.

## Speech Bubble Types

| Type | Shape | Use | Prompt Direction |
|---|---|---|---|
| Normal speech | Rounded oval | Regular dialogue | `speech bubble` |
| Thought | Cloud | Inner monologue | `thought bubble, cloud shape` |
| Shout | Burst/jagged | Anger, loud sound | `burst speech bubble, jagged` |
| Whisper | Dashed oval | Low voice | `dashed speech bubble, whisper` |
| Electronic voice | Rectangle | Phone, speaker, AI | `rectangular speech box` |
| Narration | Caption box | Time, place, commentary | `narration box, caption` |

## Bubble Placement Rules

- Follow reading order: top-to-bottom, left-to-right for Korean horizontal layouts.
- Bubble tails point to the speaker's mouth or source.
- Do not cover faces, hands, key action, or visual punchlines.
- Use at most 3 bubbles per panel; split the panel if more are needed.
- The upper third of a panel is often the easiest bubble area.

## Korean Sound Effects

| Category | Examples | Use |
|---|---|---|
| Impact | 퍽, 쾅, 팍, 둥, 탁 | Punch, collision, explosion |
| Movement | 슈웅, 휙, 스윽, 콰콰콰 | Running, flying, spinning |
| Emotion | 두근두근, 울컥, 싸늘, 찌릿 | Tension, emotion, fear |
| Environment | 우르르르, 콸콸, 사각사각 | Thunder, water, wind |
| Daily | 딸깍, 끼이익, 뚝뚝, 찰칵 | Doors, brakes, drops, camera |
| Silence | 조용..., 쥐 죽은 듯 | Awkward or tense silence |

Sound effect design:

- Size equals volume.
- Tilt equals speed.
- Color implies character: red for impact, blue for cold, yellow for electricity.
- Place sound effects near the sound source.
- Use a font style distinct from normal dialogue.

## Show, Don't Tell

| Avoid Saying | Show Instead |
|---|---|
| "나 정말 화가 나!" | clenched fist, gritted teeth, close-up |
| "여기가 위험한 곳이야" | cracked walls, warning lights, stains |
| "시간이 많이 흘렀어" | seasons, grown tree, rust |
| "그녀를 좋아하게 됐어" | repeated glances, blush |
| "이건 소중한 물건이야" | careful trembling hands |

## Emotion Codes

| Emotion | Visual Code |
|---|---|
| Anger | vein marks, shaking frame, red background |
| Sadness | rain/snow overlay, dark tone, drooping posture |
| Joy | bright tone, petals/light, open smile |
| Fear | wide eyes, darkness, long shadows |
| Surprise | white pupils, halo, sweat drop |
| Tension | narrow panels, close-up sequence, contrast |

## Text/Image Density

| Scene Type | Text Ratio | Image Ratio |
|---|---|---|
| Action | 10-20% | 80-90% |
| Daily dialogue | 40-50% | 50-60% |
| Emotion | 10-20% | 80-90% |
| Explanation/flashback | 30-40% | 60-70% |
| Comedy | 30-40% | 60-70% |
